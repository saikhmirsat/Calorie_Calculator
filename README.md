</h4>

<b>Features</b>: User side and Admin side

<b>Frontend</b>: HTML, CSS, JavaScript, React, Chakra UI, and many other libraries <br/>
<b>Backend</b>: MongoDB, Node, Express
<b>Database</b>: MongoDB
